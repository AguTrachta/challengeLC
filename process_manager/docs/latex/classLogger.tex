\doxysubsection{Detailed Description}
A class for logging action, error, and warning messages. 

The {\ttfamily \doxylink{classLogger}{Logger}} class provides methods to log messages of different severity levels (action, error, and warning). It uses the {\ttfamily spdlog} library to handle log message formatting and output. The logger writes messages to a log file and also allows displaying the most recent log entries.
