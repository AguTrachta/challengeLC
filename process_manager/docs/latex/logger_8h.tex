\doxysubsection{Detailed Description}
Provides a logging class for action, error, and warning messages. 

This file defines the {\ttfamily \doxylink{classLogger}{Logger}} class, which is responsible for logging action, error, and warning messages. The class uses the {\ttfamily spdlog} library to log messages to a file. It also provides functionality to display the most recent logs.
