\doxysubsection{Detailed Description}
A class for monitoring system resources such as CPU and memory. 

The {\ttfamily \doxylink{classResourceMonitoring}{Resource\+Monitoring}} class enables the monitoring of system resources like CPU and memory usage. The class supports real-\/time, parallelized monitoring, with the ability to start and stop the monitoring process via user input.
