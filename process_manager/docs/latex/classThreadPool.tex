\doxysubsection{Detailed Description}
A class for managing a pool of worker threads to execute tasks concurrently. 

This class creates a fixed number of threads that wait for tasks to be added to a queue. Once a task is added, the threads will execute the task concurrently. The pool can be stopped gracefully and can wait until all tasks are completed before destruction.
