\doxysubsection{Detailed Description}
A class responsible for parsing commands from an input string. 

The {\ttfamily \doxylink{classCommandParser}{Command\+Parser}} class provides a method to parse a command string into a {\ttfamily \doxylink{structParsedCommand}{Parsed\+Command}} structure, which consists of the command\textquotesingle{}s name and its arguments.
