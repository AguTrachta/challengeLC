Detailed Description

A class for managing and interacting with system processes.

The ProcessManager class provides an interactive command-line interface for managing system processes. It supports commands such as listing processes, monitoring system resources, killing processes, viewing logs, and displaying help information. The program runs in an interactive loop to accept user commands.
